Ejercicio 1

Crea un menú onde o usuario poida escoller entre a seguintes accións: a.Cargar alumnos. b. Eliminar un alumno. c. Agregar nota a un alumno. d. Borrar nota a un alumno e. Sair Para a realización do exercicio deben terse as seguintes consideracións. • Só se pode usar arrays para gardar os alumnos e as súas notas. O normal é usar un array bidimensional. • Só se sae do menú se se pulsar a opción saír.
